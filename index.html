About

HI! I'm Laurie. A Front End Web Developer based in Kitchener, Ontario. Before coding I was a Department Head who managed a team remotely. During this time I learned that communication and good relationships are key to success. I love exploring green spaces, hitting the rink (on wheels!), or sitting down with great cup of coffee and a good book. My favourite part of being a FED is seeing a project from start to finish. Finding opportunities for re-usable code is so satisfying! Now that I’m done bootcamp, I’m excited to continue building on my Javascript and React skills. I’m also interested in learning more about Web Accessibility and Node.
